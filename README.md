# College-Event-Feedback-Analysis
College events  and collect valuable student feedback. This project analyzes student satisfaction survey data to extract meaningful insights using data analytics and NLP techniques.  The goal is to understand student sentiment, satisfaction levels, and improvement areas to help institutions enhance future campus events and activities.


#Objectives

Analyze student satisfaction ratings (1–5 scale)

Perform sentiment analysis on textual feedback

Visualize trends and patterns in student opinions

Identify key areas for improvement based on feedback

Convert raw survey data into actionable insights


#Dataset

Source: Kaggle – Student Satisfaction Survey

The dataset represents feedback collected from students regarding academic activities and college events, including ratings and comments.

 Format: CSV
Type: Rating-based + Textual feedback



#Tools & Technologies

Google Colab – Cloud-based notebook environment

Python

pandas – Data cleaning & manipulation

matplotlib / seaborn – Data visualization

TextBlob – Sentiment analysis (NLP)

✔ Data cleaning and preprocessing

#Visualizations Included

Satisfaction rating distribution

Sentiment analysis (pie chart)

Department-wise / category-wise satisfaction

Feedback trend insights

#Sample Insights

Majority of students expressed positive sentiment toward events

Workshops and seminars received higher satisfaction ratings

Negative feedback commonly related to infrastructure and scheduling

Improving facilities and event organization can increase satisfaction
